# Technology Stack

## Automation & Scripting
- **Bash:** Primary scripting language for Linux and macOS environments. Adheres to POSIX standards for maximum portability.
- **PowerShell:** Primary scripting language for Windows environments. Utilizes standard PowerShell conventions for system-level automation.

## Utility & Tooling
- **Dart:** Used for utility logic, specifically for hydrating script templates and managing payload encoding (Base64).
- **Git:** The core dependency, utilized for its advanced `worktree` and `bare` repository features to manage the Flutter SDK.

## Target Platform
- **Flutter SDK:** The primary software development kit managed by this project.
